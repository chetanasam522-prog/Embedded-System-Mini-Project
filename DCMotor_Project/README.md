## Description
This project demonstrates how to control a DC motor using an 8051 microcontroller.
The motor is controlled through a motor driver circuit.

## Components Used
- 8051 Microcontroller
- DC Motor
- Motor Driver (L293D / Relay)
- Power Supply

## Working Logic
- Port pin is used to control motor direction.
- Logic HIGH rotates motor clockwise.
- Logic LOW stops the motor.
